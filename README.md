## What is GitHat?

GitHat is an auth and hosting platform that replaces Clerk and Vercel. Pre-built sign-in UI, JWT tokens for your API, and deploy at `slug.githat.io`. Bring your own PostgreSQL, MySQL, or MongoDB.

```bash
npx create-githat-app my-app
cd my-app && npm run dev
```
